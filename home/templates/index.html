Semantics-based Indexing of Historical Entities

Welcome to the prototype for my MSc Project: Semantics-based Indexing of Historical Entities

This is not a fully formed web site by any means.

The site demonstrates the following:

  • A basic python - flask website with blueprints
  • Simple login to restrict access to certain features
  • Querying for persons across a selection of SPARQL endpoints:
    • British Museum
    • Archives Hub
    • British National Bibliography
    • DBPedia (Wikipedia)
    • ... and more
  • Basic list of results
  • Discarding results
  • Adding new sparql endpoints.
  • Exploring results in three ways:
    • Compare two names to see if they are the same person.
    • Pull in extra information where one of our resources exerts a 'sameas' relationship to an another resource
    • Building a semantic display of results (identifying the different 'types' of entity)

Code for the prototype is available on GitHub: https://github.com/geekscruff/peoplesparql
