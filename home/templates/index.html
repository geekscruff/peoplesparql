People Sparql

Welcome to the working prototype site for my MSc Project: Semantics-based Indexing of Historical Entities

This site demonstrates the following:

  • A basic python - flask website with blueprints
  • Simple login to restrict access to certain features
  • Querying a selection of SPARQL endpoints using the SPARQLWrapper python library
    • British Museum
    • Archives Hub
    • British National Bibliography
    • DBPedia
    • ... and more
  • Querying for personal names across those endpoints, return a basic list of results
  • Discarding results
  • Adding new sparql endpoints (note, endpoints vary *a lot* so this does not always work).

Code for the prototype is available on GitHub: https://github.com/geekscruff/peoplesparql
